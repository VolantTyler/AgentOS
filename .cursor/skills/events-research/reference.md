# Events research — reference hints

Use as a **starting checklist**, not exhaustive. Prefer **current** official pages over this list if links rot.

## Local discovery (NYC / NJ)

- Meetup.com — search NYC + New Jersey tech; filter by date and topic (AI, agents, JavaScript, product).
- Luma.co — common for indie tech events in NYC.
- Eventbrite — tech / startup / AI filters; verify organizer identity.
- Major venues / series — **Brooklyn** / **Manhattan** AI meetups; **Bell Works**, **NJ Tech Meetup** style groups.

## Online / hybrid (agent orchestration)

- Official conference sites and **schedule** pages (workshops > keynotes for Tyler’s goals).
- **AI Engineer** (conference / community) — virtual options when offered.
- **LangChain / LangGraph**, **LlamaIndex**, **Anthropic**, **Google AI / Cloud** — webinars with agent/tooling focus (verify topic in agenda).
- **MCP** — modelcontextprotocol.io announcements; vendor blog posts with event CTAs.
- **GitHub / Discord** announcements for frameworks Tyler uses (Hermes, Antigravity-adjacent communities, etc.) — treat as **unverified** until cross-checked on an official calendar.

## Red flags

- No date, no timezone, no venue, registration only via DM.
- “Exclusive” paywalled lists with no public agenda.
- Scraped third-party calendars that duplicate and distort times.
