## Executive summary

- **Gemini (core):** **`gemini-2.0-flash*` family shutdown executed June 1** per official changelog—calls to retired IDs fail until migrated to **`gemini-3.5-flash`** or **`gemini-3.1-flash-lite`**; **May 28** GA for **`gemini-3.1-flash-image`** / **`gemini-3-pro-image`** plus **video-to-image** on flash-image and **June 25** deprecation for image preview models—**upgrade/migrate** any OpenClaw or portfolio routes still on 2.0 IDs; **Interactions API** legacy removal **June 8** remains the next hard deadline *(May 26 default was covered in the prior digest—still actionable)*.  
- **Hermes (core):** **`v0.15.0` “Velocity Release” (May 28)** plus same-day **`v0.15.1`** / **`v0.15.2` hotfixes**—massive **`run_agent.py` refactor**, kanban/swarm platform, **promptware defense**, Bitwarden secrets, **`session_search` rebuild**, and dashboard loopback **401 reload-loop fix**—**monitor → upgrade** if Tyler experiments beyond dabbling; pairs with **OpenClaw Hermes auth-profile migration** from **`v2026.5.26`**.  
- **OpenClaw (core):** **`v2026.5.31-*` pre-releases (May 31–June 1)** ship **Skill Workshop** (governed skill proposals + Control UI), **Workboard** orchestration, **Codex app-server** reliability fixes, SQLite-backed channel/plugin state, and **MiniMax M3**—**monitor → upgrade** on gateway cadence; **`v2026.5.26` security wave is pre-window headline material**—do not re-triage unless still unpatched.  
- **Cursor (core):** **3.6 (May 29)** introduces **Auto-review** run mode—allowlist → sandbox → LLM classifier for Shell/MCP/Fetch—with **`permissions.json`** steering—**monitor / policy**; classifier is **non-deterministic** per Cursor forum guidance, not a strict security boundary.  
- **Codex (core):** **`0.136.0` (June 1)** adds **Amazon Bedrock** provider, **`/archive`** sessions, app-server **`--stdio`**, remote **API-key registration**, and **command-safety hardening**; **`0.135.0` (May 28)** adds richer **`codex doctor`**, named permission profiles, Vim text-objects—**monitor → upgrade** on OpenClaw Codex pins and local CLI.  
- **ChatGPT (core):** **June 1** **U.S. job search** (Indeed/Upwork/Appcast-sourced) + global **resume formatting**; **May 28** **GPT-5.5 Instant** style update (**canvas removed** from 5.5 Instant/Thinking—writing/code blocks in chat) and **ChatGPT-only retirements** announced for **o3 (Aug 26)** and **GPT-4.5 (June 27)**—**monitor** model-picker habits if Tyler uses legacy models.  
- **Antigravity (core):** No Antigravity IDE/platform release dated **May 26–June 2** verified; **June 18, 2026** consumer **Gemini CLI → Antigravity CLI** cutoff remains **16 days out**—plan terminal/harness migration *(May 19 blog; still actionable)*.  
- **Laravel (non-core):** **`v13.13.0` (June 2)**—routine 13.x fixes plus **`Bus::bulk()`**, Cache attribute memoization, HTTP client PSR adapter—**monitor** on app cadence.
